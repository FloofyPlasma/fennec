# FenneC

# This is considered an abandoned project, please see [llvm-project-fox32](https://github.com/floofyplasma/llvm-project-fox32) instead!!

**FenneC** is a work-in-progress C compiler for [fox32](https://github.com/fox32-arch/)

> [!NOTE]
> This does **NOT** compile to binary, you must use [fox32asm]([https://g](https://github.com/fox32-arch/fox32asm)).
